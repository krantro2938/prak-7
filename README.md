# HealthyFood - Healthy Food Delivery Service

## Project Theme
**Healthy Food Delivery Service** - A minimalist landing page for a service that delivers balanced, healthy meals daily.

## CSS Framework Used
**Bulma** (v1.0.4) - A modern CSS framework based on Flexbox, providing:
- Responsive grid system
- Pre-styled components
- No JavaScript dependencies (CSS-only)
- Easy customization
- Clean, minimal design

## Features

### 1. Fixed Header
- Logo with leaf icon
- Navigation menu (About, Benefits, Pricing, Contacts)
- CTA button "Order Now"
- Responsive mobile menu (burger)

### 2. Hero Section
- Main headline with gradient background
- Subtitle describing the service
- Two CTA buttons: "Order Trial Day" and "How It Works"
- Decorative SVG illustration

### 3. About Section
- Service description
- List of key benefits with checkmark icons
- Decorative image

### 4. Features Section (Card Grid)
- 6 feature cards with icons:
  - Health Benefits
  - Save Time
  - Fresh Products
  - Free Delivery
  - Nutritionist Support
  - Eco-Packaging
- Hover effects on cards

### 5. How It Works Section
- 4-step process visualization
- Icons and descriptions for each step

### 6. Pricing Section
- 3 pricing plans: Basic, Standard, Premium
- "Popular" badge on recommended plan
- Feature lists for each plan
- CTA buttons

### 7. Order Section
- Contact form (Name, Phone, Email)
- Large CTA button
- 50% discount offer for trial day

### 8. Footer
- Company information
- Social media links (Facebook, Instagram, Telegram, WhatsApp)
- Navigation links
- Contact information (phone, email, address, hours)
- Copyright

## Responsive Design

The landing page is fully responsive:
- **Desktop**: Full navigation, side-by-side layouts
- **Tablet**: Adjusted grid columns, readable text
- **Mobile**: Burger menu, stacked layouts, touch-friendly buttons

## Accessibility Features

- Semantic HTML structure
- ARIA labels for interactive elements
- Keyboard navigation support (Tab, Enter)
- Sufficient color contrast
- Focus states for interactive elements
- Alt text for images
