# Проект: Лендинг "ZenFocus"

## Описание
Этот проект представляет собой минималистичный лендинг для приложения по медитации и осознанности "ZenFocus". Страница разработана в рамках Практического занятия №7.

## Технологии
* **HTML5**
* **CSS Framework:** Bulma (версия 1.0.0)
* **Icons:** Font Awesome

## Структура
1.  **Header:** Навигация и логотип.
2.  **Hero:** Приветственный экран с призывом к действию (CTA).
3.  **About:** Информация о приложении.
4.  **Features:** Карточки с ключевыми функциями.
5.  **Footer:** Контактная информация и ссылки.

## Как запустить
Открыть файл `index.html` в браузере или развернуть через GitHub Pages.
